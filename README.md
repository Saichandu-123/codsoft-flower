# codsoft-flower
 🚢 Titanic Survival Prediction

📌 Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning.
The model is trained on the famous Titanic dataset and uses passenger details such as age, gender, class, and fare to make predictions.
The goal of this project is to demonstrate **data preprocessing, visualization, and classification using Python and Scikit-learn**.
 📊 Dataset
The dataset used in this project contains information about passengers on the Titanic.
Main Features:
* PassengerId
* Pclass (Passenger Class)
* Sex
* Age
* SibSp (Number of siblings/spouses aboard)
* Parch (Number of parents/children aboard)
* Fare
* Embarked
* Survived (Target Variable)
Dataset Source: Kaggle Titanic Dataset
🛠️ Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* VS Code
⚙️ Project Workflow
1. Import required libraries
2. Load Titanic dataset
3. Data cleaning and preprocessing
4. Exploratory Data Analysis (EDA)
5. Feature selection
6. Train-test split
7. Model training using Machine Learning
8. Model evaluation
9. Survival prediction
 🤖 Machine Learning Model
The project uses classification algorithms such as:
* Logistic Regression
* Decision Tree (optional)
* Random Forest (optional)
The model is trained to predict whether a passenger survived or not.
📈 Output
The model outputs:
* Prediction of survival (0 = Not Survived, 1 = Survived)
* Accuracy score
* Classification report
